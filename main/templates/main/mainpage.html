🦁 2주차 Django 이야기 🦁

    🎀 Template 🎀

  • mainpage.html 페이지 파일 만들어서 첫 번째 main 화면을 구성한다
  • Views.py를 작성한다: 원하는 url을 mainpage가 보여지게 한다
    • 작성형식: def mainpage(request): return render(request, 'main/mainpage.html')
  • urls.py를 작성한다: url에 해당하는 데이터를 연결하는 것
    • main 앱에서 view 함수를 불러온다: from main import views
    • urlpatterns=[ ] 안에 경로 추가한다: path('', views.mainpage, name="mainpage")

    🎀 Template 🎀

  • nav 부분에 페이지 이름 넣어 링크해야 링크가 걸림.
  • 템프릿 변수도 사용 가능
  • 반복문, 조건문도 사용 가능

    🎀 Template 상속과 nav bas 분리🎀

  • project 폴더 내 templates 폴더 생성 후 그 안에 base.html 만들고 그 안에 반복되는 코드를 이동시킴
    • block content 와 endblock 이용. 페이지들에는 그 위에 extends base.html 을 적어주기. 베이스파일은 따옴표 안에.
  • project > settings.py에서 import os 하고 DIRS 뒤에 내용 추가
  • templates 폴더 내에 shared 폴더 생성 후 그 안에 _navbar.html 생성 후 base.html에 포함된 nav영역을 navbar.html로 옮기기. 분리했으니 base.html에 상속시켜야함.

    🎀 Static 🎀

  • project 폴더에 static 폴더 생성하고, 그 안에 css, images폴더 생성. css안에는 project.css 생성
  • load static 과 img src 이용하여 넣기
  • settings.py에서 경로 설정
